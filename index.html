operations

a = b; – assign value b to a

function (arguments) { ... } – define a function

a.b(); – call function b on element a

some useful variables

document – representation of the whole page

document.body – body of the page

element.className – value of the class attribute of the element

EXERCISES

  1. Create an empty <script> block under your buttons
  2. Create a CSS rule for the class .light that changes the background color to a light color of your choice and text color to a contrasting color of your choice
  3. Create a CSS rule for the class .dark that changes the background color to a dark color of your choice and text color to a contrasting color of your choice

EXERCISES

  1. In the <script> block write a script that will add event listener on the first button for an event "click", and the executed function will change the class of the <body> element to .light. (Hint: use the button id to retrieve it!)
  2. In the same block add another script, which will add event listener on the second button for the same event, but will change the class of the <body> element to .dark
